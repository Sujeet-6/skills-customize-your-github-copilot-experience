
# 📘 Assignment: Hangman Game Challenge

## 🎯 Objective

Build the classic Hangman word-guessing game using Python strings, loops, conditionals, and user input.

## 📝 Tasks

### 🛠️ Game Setup and Word Selection

#### Description
Create the main game logic and select a hidden word for the player to guess.

#### Requirements
Completed program should:

- Choose a random word from a predefined list.
- Display the hidden word as blanks, e.g. `_ _ _ _ _`.
- Show the current word progress after each guess.
- Allow the player to guess one letter at a time.

### 🛠️ Guess Tracking and End Conditions

#### Description
Track incorrect guesses and determine when the game is won or lost.

#### Requirements
Completed program should:

- Keep track of correct and incorrect letter guesses.
- Update the displayed word progress for correct guesses.
- Decrease remaining attempts for incorrect guesses.
- End the game when the player guesses the word or uses all attempts.
- Print a victory message when the player wins.
- Print a defeat message and reveal the word when the player loses.
